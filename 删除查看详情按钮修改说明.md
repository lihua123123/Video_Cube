# 删除知识卡片侧边栏"查看详情"按钮

## 📋 修改说明

用户希望删除视频页面右侧知识卡片侧边栏中的"查看详情"按钮,因为该按钮没有实际功能。

## 🔧 修改内容

### 文件: `Frontend/src/components/KnowledgeCardDisplay.vue`

#### 1. 删除模板中的卡片底部操作区域 (第 44-59 行)

**删除前**:
```vue
<!-- 卡片底部操作 -->
<div class="card-footer" v-if="expandedCardId === card.id">
  <button class="view-details-btn" @click="viewCardDetails(card)" title="查看详情">
    <svg width="14" height="14" viewBox="0 0 24 24" fill="currentColor">
      <path d="M12 4.5C7 4.5 2.73 7.61 1 12c1.73 4.39 6 7.5 11 7.5s9.27-3.11 11-7.5c-1.73-4.39-6-7.5-11-7.5zM12 17c-2.76 0-5-2.24-5-5s2.24-5 5-5 5 2.24 5 5-2.24 5-5 5zm0-8c-1.66 0-3 1.34-3 3s1.34 3 3 3 3-1.34 3-3-1.34-3-3-3z"/>
    </svg>
    查看详情

  <div class="card-info">

  </div>
```

**删除后**:
- 整个 `.card-footer` 区域被完全移除
- 卡片内容区域直接结束

#### 2. 删除相关CSS样式 (约第 580-630 行)

删除了以下CSS类:
- `.card-footer` - 底部容器样式
- `.view-details-btn` - 查看详情按钮样式
- `.view-details-btn:hover` - 按钮悬停效果
- `.card-info` - 卡片信息容器
- `.ai-tag` - AI生成标签样式
- `.content-type-tag` - 内容类型标签样式

#### 3. 简化事件处理逻辑 (第 271-276 行)

**修改前**:
```javascript
} else if (!target.closest('.card-footer')) {
  // 如果不是点击链接或底部按钮，则查看详情
  viewCardDetails(card);
}
```

**修改后**:
```javascript
} else {
  // 如果不是点击链接，则查看详情
  viewCardDetails(card);
}
```

**说明**: 由于底部按钮已删除,不再需要检查 `.card-footer`

## ✅ 保留的功能

虽然删除了"查看详情"按钮,但以下功能仍然保留:

1. ✅ **时间点击跳转**: 点击时间可以跳转视频进度
2. ✅ **卡片展开/收起**: 点击卡片头部可以展开或收起内容
3. ✅ **链接点击**: 卡片内容中的链接仍然可以点击
4. ✅ **点击卡片查看详情**: 点击卡片内容区域仍会触发 `cardClick` 事件

## 🎯 用户体验改进

### 改进点

1. **界面更简洁**: 删除了无实际功能的按钮,减少视觉干扰
2. **减少误操作**: 用户不会点击无效的"查看详情"按钮
3. **空间利用**: 卡片底部空间被释放,内容区域可以更大

### 操作流程

```
用户查看知识卡片:
1. 看到右侧卡片列表
   ↓
2. 点击时间可跳转视频 ✅
   ↓
3. 点击标题可展开/收起 ✅
   ↓
4. 点击内容查看详情 ✅
   ↓
5. 点击链接打开链接 ✅
```
