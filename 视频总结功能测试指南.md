# 🎬 视频总结功能 - 快速测试指南

## 🚀 快速开始

### 1. 确保服务运行
```powershell
# 终端1: 启动后端
cd Backend
npm start

# 终端2: 启动前端
cd Frontend
npm run dev
```

### 2. 访问页面
打开浏览器访问: `http://localhost:5173`

### 3. 加载视频
- 方法1: 从视频库选择一个视频
- 方法2: 输入视频URL并点击"加载视频"

### 4. 测试总结功能
当视频加载成功后,在视频播放器下方会看到"视频总结"组件。
