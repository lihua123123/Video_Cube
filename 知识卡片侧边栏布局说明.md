# 知识卡片侧边栏布局说明

## 📋 功能概述

将知识卡片显示区域改为右侧边栏形式,提供更好的视觉体验和交互体验。

## ✨ 主要改进

### 1. 布局结构
- **左右分栏布局**: 视频区域在左侧,知识卡片在右侧
- **固定宽度侧边栏**: 桌面端宽度为 380px,大屏幕(≥1400px)为 420px
- **响应式设计**: 
  - 平板/手机端(<1024px): 侧边栏移到下方,横向布局
  - 小屏幕(<768px): 进一步优化间距和字体

### 2. 视觉设计
- **渐变色头部**: 紫色渐变(#667eea → #764ba2)显示"相关知识"标题
- **卡片计数**: 实时显示当前可见的知识卡片数量
- **浅灰背景**: 侧边栏背景色为 #f8f9fa,与主内容区区分
- **阴影效果**: 左侧阴影(-2px 0 8px rgba(0,0,0,0.05))增强层次感

### 3. 交互优化
- **独立滚动**: 侧边栏内容可独立滚动,不影响主内容区
- **收起/展开**: 保留原有的卡片展开收起功能
- **点击查看详情**: 保留原有的详情查看功能

## 🎨 样式特点

### 侧边栏容器
```css
.knowledge-cards-sidebar {
  width: 380px;                    /* 固定宽度 */
  background: #f8f9fa;             /* 浅灰背景 */
  border-left: 1px solid #e0e0e0; /* 左侧边框 */
  box-shadow: -2px 0 8px rgba(0, 0, 0, 0.05); /* 阴影 */
}
```

### 头部样式
```css
.sidebar-header {
  padding: 20px;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
}
```

### 内容区域
```css
.sidebar-content {
  flex: 1;
  overflow-y: auto;  /* 独立滚动 */
  padding: 16px;
}
```

## 📱 响应式断点

### 大屏幕 (≥1400px)
- 侧边栏宽度: 420px
- 更宽敞的卡片显示空间

### 平板 (≤1024px)
- 侧边栏移至底部
- 最大高度: 400px
- 横向布局

### 手机 (≤768px)
- 侧边栏最大高度: 300px
- 减少内边距优化空间
- 字体大小适配

### Flexbox 布局
```css
.main-content-wrapper {
  display: flex;           /* 使用 Flexbox */
  flex: 1;

```

## 🎯 用户体验提升

### 1. 视觉清晰度
- ✅ 知识卡片始终可见,无需滚动查找
- ✅ 视频和卡片同屏显示,便于对照学习
- ✅ 紫色渐变头部吸引注意力

### 2. 操作便利性
- ✅ 独立滚动,互不干扰
- ✅ 固定位置,易于访问
- ✅ 响应式设计,适配各种设备

### 3. 空间利用率
- ✅ 充分利用宽屏显示器的横向空间
- ✅ 小屏幕自动调整为纵向布局
- ✅ 避免过多留白

## 📊 对比说明

### 改进前
- 知识卡片在视频下方
- 需要向下滚动才能看到
- 占用大量纵向空间

### 改进后
- 知识卡片在视频右侧
- 始终可见,无需滚动
- 充分利用横向空间
- 更符合"一边看视频一边学习"的使用场景

## 🚀 使用建议

1. **桌面端**: 推荐使用 ≥1400px 宽度的显示器,获得最佳体验
2. **平板端**: 竖屏使用时,侧边栏会自动移到底部
3. **手机端**: 可能需要滚动查看卡片,建议横屏使用

2. **侧边栏折叠**: 添加完全隐藏/显示侧边栏的功能
